# Assets Folder

Put your own portfolio media files in this folder.

Examples:

- `assets/photo-1.jpg`
- `assets/photo-2.png`
- `assets/poster-cover.jpg`

Then update `script.js` to point to those files.

Example:

```js
imageUrl: "assets/photo-1.jpg";
```

Note:

- Videos on portfolio sites are usually added with YouTube or Vimeo embed links.
- Photos can be local files stored in this folder.
